Introduction

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our Car parking API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

To authenticate requests, include an Authorization header with the value "Bearer {TOKEN}".

All authenticated endpoints are marked with a requires authentication badge in the documentation below.

You can retrieve your token by visiting your dashboard and clicking Generate API token.

Register

User registration

Example request:
curl --request POST \
    "http://localhost/api/v1/auth/register" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"name\": \"cufwydldpymzmjec\",
    \"email\": \"aliza47@example.net\",
    \"password\": \"nostrum\"
}"

const url = new URL(
    "http://localhost/api/v1/auth/register"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "name": "cufwydldpymzmjec",
    "email": "aliza47@example.net",
    "password": "nostrum"
};

    method: "POST",

Create Zone

Create a zone

Example request:
curl --request POST \
    "http://localhost/api/v1/zones" \
    --header "Content-Type: multipart/form-data" \
    --header "Accept: application/json" \
    --form "name=eqddksajchthntpimnm"\
    --form "price_per_hour=21"\
    --form "image=@C:\Users\Win 10\AppData\Local\Temp\phpC3C0.tmp"

    "http://localhost/api/v1/zones"
);

const headers = {
    "Content-Type": "multipart/form-data",
    "Accept": "application/json",
};

body.append('name', 'eqddksajchthntpimnm');
